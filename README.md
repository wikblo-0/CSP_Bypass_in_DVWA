This repository stores the payloads used in the penetration testing of the Content Security Policy (CSP) Bypass vulnerability in Damn Vulnerable Web Application (DVWA).

- low_security_level.js stores payloads used on the low security level.

- medium_security_level.html stores payloads used on the medium security level.

- high_security_level.js stores payloads used on the high security level.

You can read more about the attack in the blog linked below, and there you will also find YouTube videos showing a screencast of each attack.

https://pentesting-dvwa.hashnode.dev/
